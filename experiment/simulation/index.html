State Elimination Algorithm

Goal: Convert NFA to Regular Expression by eliminating intermediate states

1

Preparation

Ensure NFA has exactly one start state and one accept state. Add ε-transitions if needed.

2

State Selection

Choose an intermediate state to eliminate (neither start nor accept state).

3

Transition Creation

For each pair of states (p,q) where p→rᵢ→q, create new transition p→q with label combining all paths through rᵢ.

4

State Removal

Remove the selected state and all its transitions from the automaton.

5

Repeat

Continue until only start and accept states remain. The transition between them is your regular expression.

Transition Formula:

R(p,q) = R(p,q) + R(p,rᵢ) · (R(rᵢ,rᵢ))* · R(rᵢ,q)

Where rᵢ is the state being eliminated
